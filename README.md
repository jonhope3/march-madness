# 🏀 March Madness 2026

A beautiful, live-updating NCAA Tournament dashboard hosted on GitHub Pages.

**Live at: [jonhope3.github.io/march-madness](https://jonhope3.github.io/march-madness)**

## Features

- 🔴 **Live Data** — Pulls real-time game data from ESPN on every page load
- 📅 **Full Tournament Schedule** — Navigate between First Four through Championship
- 🏟️ **Game Details** — Teams, seeds, records, venues, locations, TV channels
- 📊 **Odds & Predictions** — Spreads, over/under, moneylines from DraftKings
- 🔮 **Predictions** — Projected winners based on betting lines
- 👤 **Player Leaders** — Season stat leaders for each team
- 🎨 **Premium Design** — Dark theme with glassmorphism, particle effects, animations
- 📱 **Fully Responsive** — Looks great on desktop, tablet, and mobile
- ⚡ **Auto-Refresh** — Refreshes every 30 seconds when live games are in progress
- 🔍 **Filtering** — Filter by live, upcoming, or completed games

## Tech Stack

- Pure HTML, CSS, and JavaScript (no frameworks)
- ESPN public API for live data
- Hosted on GitHub Pages (static site)
